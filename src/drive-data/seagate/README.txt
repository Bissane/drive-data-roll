These scripts are provided to the San Diego Super Compute (SDSC) center 'as is' by Seagate.
These scripts collect data about IO activity.
These scripts collect no user data.
With the exception of the tool with the word 'Private' in its name, all can be shared publically.
The tool with 'Private' in its name is provided under NDA and must be protected from escaping the control of SDSC.
